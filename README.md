# Mission3

The program helps a local food bank manage its inventory of donated food.

## Features
- **Add Food Items** – Name, Category, Quantity, Expiration Date  
- **Delete Food Items** – Remove an item by name  
- **Print Inventory** – See all current food items  
- **Exit Program**

## How to Run
1. Open the project in **Visual Studio** or **VS Code**.  
2. Compile and run `Program.cs`.  
3. Use the menu options to manage the food inventory.

## Notes
- Quantity must be a number ≥ 0 (invalid input defaults to 0)  
- Expiration date format: `yyyy-mm-dd` (invalid input defaults to today)  
- All food items are stored in memory (not saved to a file)
